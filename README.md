# blackout

A browser extension to make blackout poetry on the interwebs.

## Instructions
- Click on the extension button.
- Select text you want to black out.
- Click on the extension button again to reset the page.

## Get Extension

Firefox:
https://addons.mozilla.org/en-US/firefox/addon/blackout-poetry/

Chrome/Brave/all Chromium-based browsers:
https://chromewebstore.google.com/detail/ffcdlpjolgglihiapammdhbdogcgjbki
